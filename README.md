# [Takuma Tech Blog](https://takumatoiyama.com/)

A personal tech blog built with Astro and deployed on Cloudflare Pages.

## Features
- Japanese / English blog posts
- Tag-based article organization
- Static site generation with Astro
- Automatic deployment through Cloudflare Pages

## Tech Stack
- Astro
- HTML / CSS / JavaScript / TypeScript
- Cloudflare Pages
- GitHub

## What I learned
- Static site deployment
- Custom domain setup
- Astro project structure
- Using AI tools while reviewing and modifying generated code
