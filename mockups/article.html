Making this website

2026-05-24 · 8 min read · Dev log

I built this blog with Astro and deployed it on Cloudflare Pages. The whole thing is static — Markdown files in src/content/blog/ become HTML at build time.

Setup

The starter template gave me a working blog in a single command. From there I rewrote the styles, added bilingual support, and connected Obsidian as my editor.

npm create astro@latest -- --template blog
cd my-astro-project
npm run dev

The dev server runs at http://localhost:4321. Hot reload picks up changes to Markdown files immediately.

Deployment

Cloudflare Pages connects directly to GitHub. Every push to main triggers a build and the new version is live within a minute.

git add .
git commit -m "Add new post"
git push origin main

That's the whole flow. No servers to manage, no databases to worry about.
